  • Multiple account types (401k, IRA, Roth, HSA, ESOP)
  • Scenario comparison (compare different strategies)

  • Export/import configurations

Scroll Stitcher

Convert a mobile screen recording of scrolling content into a single full-length screenshot.

  • Drag-and-drop video upload (MP4, MOV, WebM)
  • Cross-correlation frame alignment — no manual input
  • Supports scroll in any direction (down, up, left, right)
  • Two-pass processing keeps memory usage low
  • Download full-resolution PNG
