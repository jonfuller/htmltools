Key Concepts:
  • Monte Carlo Simulation: Running thousands of scenarios with random market returns to model uncertainty
  • Log-Normal Distribution: Models how investment returns actually behave (can't go below -100%, but can grow infinitely)
  • Percentiles: 10th percentile = pessimistic outcome, 50th (median) = expected, 90th = optimistic
  • Inflation Adjustment: All results shown in today's dollars for meaningful comparison
